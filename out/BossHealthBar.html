Class: BossHealthBar

BossHealthBar(boss)

Manages and draws a health bar for a boss enemy.

Constructor

new BossHealthBar(boss)

Creates a new BossHealthBar instance for the specified boss.
Parameters:
Name Type Description
boss Object The boss object for which the health bar will be drawn. The boss object should have the following properties: - {number} x: The x-coordinate of the boss on the canvas. - {number} y: The y-coordinate of the boss on the canvas. - {number} width: The width of the boss in pixels. - {number} height: The height of the boss in pixels. - {number} health: The current health of the boss. - {number} maxHealth: The maximum health of the boss.
Source:
  • models/boss-health-bar.class.js, line 4

Methods

draw(ctx)

Draws the health bar onto the specified canvas context. The bar is rendered in two colors: - Dark red for the background (total possible health). - Lime green for the boss's current health.
Parameters:
Name Type Description
ctx CanvasRenderingContext2D The canvas rendering context.
